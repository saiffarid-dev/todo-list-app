# 📝 To-Do List App

A modern and responsive To-Do List application built using **HTML, CSS, and JavaScript**. This project allows users to add, complete, delete, and save tasks using the browser's Local Storage.

---

## 🚀 Features

- ✅ Add New Tasks
- ✅ Mark Tasks as Completed
- ✅ Delete Tasks
- ✅ Local Storage Support
- ✅ Responsive User Interface
- ✅ Clean and Modern Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM)
- Local Storage API
